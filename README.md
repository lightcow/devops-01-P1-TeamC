# proejct1  
### youtube 동영상과 취미 용품 구매 목록
  
## 작성된 api 목록

GET - 전체 영상 보기  
GET - 특정 영상 보기-구매 가능한 물건 목록  
GET - 태그로 영상 검색  
POST - 장바구니 아이템 담기  
PUT - 장바구니에 아이템 수량 수정  
DELETE - 장바구니 아이템 지우기  
  
## Available Scripts
